📖 關於演算法雙軌制（買斷 vs 協議合建）

本系統支援不動產估價技術規則之「土地開發分析法」與業界實務之「成本加成等價交換法」。

買斷模式: V = { S / (1 + R) - (C + M) × [ 1 + (WACC × n/2) ] } / (1 + WACC × n)
合建模式: Dev_Value = (C + M + Build_Interest) × (1 + Cost_ROI)
  • 買斷模式 (算底價)：利潤目標為全案總銷之 X%。需扣除購地之土地融資利息，對地主條件較為嚴苛。
  • 合建模式 (算比例)：利潤目標切換為「建商投入成本之加成報酬 (Cost ROI)」。無土地融資利息扣減。
  • WACC (綜合資金成本率)：混合資金機會成本，(自有比例 × 報酬率) + (融資比例 × 利率)。
  • 等價交換法：依據合建模式推算出之建商應分回價值 (Dev_Value)，與剩餘之土地價值 (V) 進行比例分配。
